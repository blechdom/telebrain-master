jsPlumb provides a way to "plumb" elements of a UI together.

This page provides a demonstration of the 'perimeter' anchor functionality in jsPlumb - anchor points that track of the perimeter of some shape or curve.

jsPlumb requires one of jQuery, MooTools or YUI3. See the documentation for a more detailed discussion of requirements.

This demonstration uses jsPlumb 1.3.13, jQuery 1.7.1 and jQuery UI 1.8.13.
